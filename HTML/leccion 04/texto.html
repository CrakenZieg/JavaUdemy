Pagina texto.html introducida como iFrame

Texto dentro de texto.html, se encuentra en otro archivo y podemos agregarlo a este mediante la etiqueta iframe.

Podemos agregar sitios Web enteros usando el atributo 'src' de la etiqueta iframe o podemos agregarlos utilizando un atributo 'name' en iframe y una etiqueta 'a' de link y dandole un atributo 'target' con el 'name' del iframe como valor
